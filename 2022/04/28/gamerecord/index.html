Feb.

+ 大逆转裁判

大逆1+2应该是玩了70h+。

准确而言二月份只玩了2，而且好像玩到了二月。21年买的，打完了1，因为是日文的，打得拖拖拉拉，中间又有实习，玩了挺久的。一开始是冲着亚双义买的游戏，他真的很帅！没有多想、禁不住诱惑，看脸就买了，并且立刻搜了一波剧透看看他结局是什么。

逆转系列说实话我都没有自己推理，基本上是为了看剧情，对着攻略打的，人很懒， 1的时候主线剧情还不太明朗，但还蛮吸引我的，1-3到英国的第一个案子我还挺喜欢，它合我的口味，恶人获胜，正义一方无能为力。因为逆转1总是成步堂获胜嘛，看到这里龙介虽然也算是辩护成功，但却是为恶人辩护，觉得挺有意思。2-1我也挺喜欢，是寿沙的主场，打到这里时还期待过后面还会有寿沙辩护的章节，不过没有出现，这一章应该是作为新手教学章节而安排的吧，但是寿沙真的很凛々しい！最精彩的肯定是2-4与2-5，埋了许久的伏笔到此时一一揭晓，这两章我也打得废寝忘食。但是为了快速推完主线，调查部分好多都没有去对话，于是错过了2-3（？）去检察官办公室调查时爱丽丝与龙介对班吉克斯的吐槽，太可惜了！

不过大逆的结局部分，总觉得是不是有些编不下去了，给出线索的方式也轻率了许多，最后看到全息投影的时候还是忍不住吐槽了，或许看到2-3的超能力电磁炮（不是）时就应该警觉了！

逆转这个系列我只打了逆转1，逆转2止步于2-2，只有大逆打完。123重置版没有重置画面（就还是原来那个画风，没有大改），而大逆的画风比较对我胃口，故事也是，玩的时候就相对快乐。逆转之前我也吐槽过好几回，打了1-5之后我就打不太下去了，第一部里我最喜欢的是1-4，记忆中1-4的内容其实和前面三章没有太大关系，不过长度刚好，里面反转也多，内容足够精彩。而1-5，时间线上应该是接在1-4之后的，但就内容而言更像是“总集篇”，而我才刚打完1-4呢，人物都还没怎么展开，千寻姐似乎也才离开了没有多久……这时候出现新的类比绫里姐妹的宝月姐妹，反而让我观感不佳。后来嗼友解释说是这个「番外篇」是之后才加的，只是按故事时间，放在了1-4的后面。当时因为打1-5实在太疲惫了——对，尽管反转很多，故事也算得上足够精彩，可我打下来就只想着“怎么还没有结束……”，于是打完了2-1之后没太大兴趣继续了，很快搁置了该系列。结束大逆后，又对逆转产生了兴趣，之后有时间了应该会继续！

+ 绯红结系

绯红结系大约是57h通关。

这个游戏发售之前我就有点兴趣因为我是万代冤大头，还只有海报的时候觉得男主很可爱，但是之后又有破晓传说，一次性买俩太奢侈了，最后还是买了破晓传说。结果这俩半斤和八两。

先说剧情，游戏分了两条线，男主结人线与女主重线，应该是想把剧情分割成两部分，两条线通了之后才能获得真相，有点类似于麦卡沃伊与杰西卡的电影《他和她的孤独情事》，问题也差不多，那就是，真的有必要拆成两个部分吗？

听说结人线做得很差，我就先玩了结人线，以免到时候吐槽这条线太多。实际上玩的时候我也是一头雾水，时常处于一种“这、这是在干什么”的状态，永远都不知道剧情是如何推进到这一步的，一回过神，男主就成了救国英雄，到下一章，他又被通缉了，什么进展啊？有人出来讲讲吗？反派角色也是。也是因为分成了两条线，但是战斗上又不能差太多，所以剧情上就是自动安排了一拨人来男主这个队，另一波人去了女主那边，理由么，可以解释，又觉得挺牵强的。之后两拨人相遇了，一言不合就开打了，打着打着，女主那队又突然撤退了……很迷惑。到了最后，男主线与女主线收束成了一条，一起对付反派。典中典的剧情又来了：反派的基友出现了，拥抱他说没想到你过得这么苦，我都不知道。最后反派被感动了，不，终于愿意停手了，牺牲了自己，创造了美好未来。END。大为震撼，立刻滴滴基友说：我去，我说什么来着，这根本就是给吧！

到了女主重线这儿，比结人线要清晰一些，合理怀疑结人线是不是硬加出来的。重的叛变有理有据，至少是能理解的，尽管她身边的人愿意和她一起叛变的理由依然比较牵强，游戏后期也没有细细讲。她这条线的行动也相对清晰，解释了一部分走结人线时看起来莫名其妙的行为。重不擅长与人交流，她不向男主解释，最开始两个人分成两路之后，男主私信她，她直接拉黑了男主，所以剧情看下来，重就像个谜语人，有时候明明可以解释，却总是被打断，演变成打一架，再到下一次。甚是无语。

而总的来说，该游戏的设定倒是不错，未来世界，人们拥有了超脑力，城市是朴素的建筑+全息投影，只有一部分人因为没有超脑力而无法看见投影，这部分人被歧视性地称为“甜椒”。角色们大都是大家族的孩子，男主结人是这个国家的创始人的后代，父亲与哥哥是议员/将军，女主是重工企业的养女，其他角色也都是如此，某制药公司的孩子，某医院企业的旁系小孩。他们都是超能力者，为了杀死无脑的怪物而服役，而这些怪物中，竟然有许多都是人为制造的，很多与他们一样的退伍军人甚至在加速死亡后被取走大脑做成了药物……这种设定很显然是想影射财阀与政治，奈何剧情太普太次，它都没怎么在我的脑子里留下印象。这一点上和破晓传说也真的很像。

多说一嘴反派卡连，这个人物是他们超能力军队的TOP1，男主小队正太鲁卡是卡连的亲生弟弟，可是呢，剧情里俩兄弟相见，卡连打鲁卡毫不心慈手软，不禁怀疑鲁卡根本是个便宜弟弟。卡连的幼驯染之一，吹雪，则作为政府的走狗，为了保护卡连自愿接受了政府的性格改造（他亲口说的），调停卡连与男主小队等等，甚至最后还是他出现拥抱卡连（如前所述），重篇，众人穿越到了50年后，卡连抱着吹雪的尸体出现，吹雪的姐姐岚质问卡连，吹雪是不是因为卡连而死。天啊，你们真的是给吧。 剧情里，卡连、吹雪与艾丽丝青梅竹马，卡连与艾丽丝互通心意，但是吹雪与艾丽丝的家族要求他们联姻。最后艾丽丝为了保护卡连而死，卡连则不断地穿越时间试图拯救艾丽丝…………与吹雪。看到此处我又开始怀疑鲁卡真的是个便宜弟弟。

虽然打正篇时疯狂吐槽，但还是很诚实地购买了一个DLC，又因为没满级也一直打不过99级卡连，于是没能看到最后一段剧情，只能视频通关一下。卡连回溯到了最初，直接让自己消失了，抹除了自己存在过的痕迹，换艾丽丝活下来，只有主角一方一共12人以及吹雪记得他，连活下来的艾丽丝都不知道卡连是谁。真的受不了了什么典型剧情 DLC是卡连的视角，讲了艾丽丝出事后他如何不断地穿越试图拯救艾丽丝，但同时又会害死男主或者女主，或者害死鲁卡，或者害死吹雪。艾丽丝变成妖异之后，他与吹雪私底下追查，终于找到了艾丽丝，然而他们发现这是个圈套，吹雪为了救卡连，选择与艾丽丝同归于尽。我大喊这就是爱——————————

Mar.

+ 三角战略

47.5h一周目完美结局通关。

对战旗没偏好，不过看到是SE的，期待了一段时间。

先说剧情，总体而言我还算喜欢三角的剧情，此处主要对比三房。类似的故事，三角写得要认真许多，相比之下三房的故事就是充满了幻想，轻松得像在做梦。三角主要是选择，剧情看着也像是切片，不过只言片语构建起来的故事与人物也更加贴近现实，每个人都有自己的考量，而不是出于什么“爱”“情谊”，比如管家本尼迪特代表“功利”，他作为tactician，建议基本上不会从什么感情角度考虑，而是从对本国的利益考虑的。同为tactician的库罗德，在这方面就像是过家家。又比如，分支选了炸人工湖之后的剧情，罗兰虽然成功收复国家但不少人民依然不满，甚至觉得帝国更好。

三角我在象上零零碎碎还写了挺多的，干脆复制粘贴一下吧。

  • 不过仔细想想罗兰也不能说软弱……？其实交出他确实是最优解，鉴于对方只是对王族下手，国王也可以代代更新，世界又不会因此终结，没得也只不过他的家人，而一家四口两个已经死了，为了自己就要牺牲大多数人。

    而且牺牲自己不是说说这么容易的，能下这样的决心也不容易，毕竟大部分人都不想死，都想活着并有起码的尊严。

    当然复仇是大多数作品的“正确”选择，又考虑到这么多“丧权辱国”（丧权辱国），才觉得罗兰是扶不起的阿斗。对普通人来说，谁领导无所谓，只要自己过得好就行了。
  • IGN评三角：In a genre filled with flamboyant and memorable characters in games like Fire Emblem, Final Fantasy, or Persona, it’s almost shocking to have a set of main characters be so aggressively dull.

    主线剧情中他们真的有机会发火闹脾气吗，都身居高位有很多要考虑而且主要cutscene都是开会……

    这里人物的塑造更多地在于他们面对每一个困境时的选择与理由，目前为止其实每个人的立场都很坚定，能看得出来他们的主要视角。加上一点点轶闻内容作为补充，玩家完全可以自己推出他们的方方面面。而且，其实也可以看出玩家个人的投射，也有很多空间让玩家去想象。

    同时我并不需要游戏像人物设定集一样把他的方方面面都写出来，原作把角色过去的所有经历、他的各种喜好等等全部列出来就失去了想象的空间。

    eg. lose temper也要看场合，不是说一定要加上这种人物才看起来更真实吧，它也有可能只是更抓马了……你看三房有些人不抓吗，三房还有学院时期三角没有。而且，高压环境下人可能会失去情绪，做一整天流水线员工看看，其实这一整天都可能没有心情起伏，至少我自己做富士康员工的那几天是真的毫无反应，去工厂之前我还在家饭来张口衣来伸手呢…………

    不过这也是我自己的理论体系的投射。

    原来罗兰是美丽废物人尽皆知。

当然三角的剧情也有不满的地方，最不满的其实就是完美结局，我为什么还要打呢，因为我贱吧，有隐藏结局一定会对着攻略打出来隐藏结局。打完之后非常不喜欢，最后就没二周目了（也有别的因素），因为也快出国了就把游戏寄给了朋友，之后可能还是会下个电子版继续二周目吧。

还不是隐藏结局的部分：最无语的剧情是有情人终成兄妹，管家对男主说，其实你是国王的亲生儿子。尽管是典型剧情，但是因为前面的部分走得我都还挺满意，所以突然来这么抓马狗血的一下还有些不适应，谁知还只是个开始，算了，就是个能走管家线的必要设定，忍了。后期古斯塔德弗也疑似降智，翻到当时的一条嘟文说：

古斯塔德弗这么谨慎多疑的人，在知道奥蕾雅是罗洁尔解放运动的领导者、她的手记被魔法锁住之后，面对手记失窃竟然这么轻视，很难相信……也看不起罗洁尔人吗？讲着自由、任人唯贤却如此；还是说纯粹不在意奥蕾雅，但她好歹也是曾经的解放运动的领导者，用魔法锁住的要传给女儿的手记怎么可能只是歧视性言论记载。

古川（？）配音的这个人本也可以再展开讲讲，后面就这样收场了也甚是可惜。他应该是典型文官，靠嘴皮子说服别人（所以大招技能还是魅惑，倒很贴切）。他并不完全支持教皇，认为要进行改革，当然，也不会认同男主，毕竟改革本国与与男主&罗兰方合作，中间还是差了很多的，而且显然他是支持本国靠盐控制其他国家的，也是认同目前教皇的这种统治的。要是可以再多讲些他就好了。

玩家作为上帝视角的人物作出看似更正确的选择要容易得多，游戏内的人物却不必如此。站在上帝视角，作为受过一定教育的现代人，看盐国的统治方式，大部分人都不会赞成，因为这不是平等，教皇所言也不过是歪理，尽管完美结局的最终章教皇的话似乎也很有道理。站在被统治者的角度，哪怕无法反驳，我们也必然会反对这种统治。然而站在统治者角度想，又觉得很好理解（尽管也知道这种做法是错误的）——因为无法兼顾所有人，面对实际存在的电车难题，你要怎么做？所以打游戏的时候反而觉得没必要苛责这些人物，不需要用我的、现代的价值观评价他们。只要解释得通都可以。

完美结局还有一点我也不是很能理解：三个国家最后还是合并成了一个，因为我玩时以为三国依旧会分治，只不过更加友好。铁国的合作者竟然甘愿成为自治区……信积拉奶。

种种原因最终导致我并不怎么喜欢该完美结局，还吐槽了好几回。

以及一点简单的战斗：选了普通难度，战斗方面，同是普通，比三房难不少，因为需要考虑的地方多，魔法的设置也好符合现实，尽管我从没应用过（但是在盐湖时被雷击过好几次）。体感最难的一章是选择不交出罗兰后守住沃荷德家，与奥罗拉战斗，为了达到完美结局，也没有使用机关，这里打了好久，打到我开始怀疑普通难度是不是给普通玩家玩的了。之后还挺顺畅的，基本都没看攻略，自己试一下就能过。团灭后保留经验等级重新开始也降低了游戏难度，所以后期也只在最后的矿场与教皇战又卡了两次。
翻到了之前的嘟：

其实我觉得三房的剧情很不怎么样，太儿戏，相比之下三角剧情好太多，好好在它尝试去认真写一个故事，每一个抉择都有现实考量。

然而，三房的细节很好，很多，很用心，人物性格多样鲜明，能够让人一下就记得住，细节也更多地在于人物的塑造。这是我喜欢三房的地方，不可否认在一个描写过于轻率的背景框架下，尽管在很多方面缺乏“描述”而显得略无说服力（比如作为tactician的库罗德，其实这方面说服力并不强），但人物依旧很鲜活。这也是它同人方面生命力旺盛的原因吧。
三角如果稍微再多一些小剧场可能人物也会更出彩吧，相对来说三角的人物关系比较固定，也不像三房那样那么多人物几乎每个人都与其他十几个人有故事可看。三角大概一开始就重在“决策”，每一个故事切片也都是让玩家做出选择。
各有优缺点吧，两者一开始的定位也有大概头绪，重点根本就不一样。

总结大概就是剧情我更喜欢三角，人物我更喜欢三房。

不过，三角的人物我还算喜欢，也正是因为它不够详细，满足了我自己创造的需求……



+ Code Realize~创世的姬君~

大约25h通关。

打这个游戏是因为，12买了桥姬回家，结果回家之后才发现，咦，我的桥姬呢？？算了，买了都买了，那就打呗。

对乙女游戏……没什么太大的兴趣，早年看薄樱鬼就不大喜欢雪村千鹤这类的女主，CR的女主也没好到哪里去。打之前先搜了波剧透，放低了对游戏的期待，顺便看了圈对几条线的评价，看到有人说：fran>lupin,impey>>>>>>>>>>>>>>san>>van，那我很高兴啊，因为我就喜欢虐的！于是高兴地打了范线，这、这就是你们说的虐吗！！这也虐？这也虐？？

范线：目睹圣杀了菲尼斯之后女主和凡竟然问都没问一下伊蒂亚到底是什么是否太过离谱，而且就这样继续安然住在一个未知者（并且从菲尼斯的遗言也很容易判断伊蒂亚应该都不是普通人类）家里，这心也太大了吧。

还有这个感情线我也真的不懂，前八章都好像没什么，除了因倍，到第八章结局时忽然就跳出了凡的脸，然后第十章女主在别人口中已经是你的小女友了……

好莫名的感情线……算了，恋爱游戏，我接受了这个设定jpg

其他几条线走得更舒服点，圣线也觉得挺抓的，走这条线感觉像是，我广告看得好好的怎么突然跳出来个电视剧插播？圣都已经是活了几千年的人了耶……算了，恋爱游戏，我接受了这个设定jpg（again

因倍，本来我是不喜欢这款人物的，但因倍线反而走得比较快乐，恋爱也更轻松愉悦，走下来觉得他们确实挺不错的！结果5条线里相对还是比较喜欢因倍线，走完之后也觉得因倍挺可爱的了。因倍线是我想象中的很传统的乙女游戏剧情……

几条男配（姑且这么称呼）线里，除了因倍，其他人的几条线，其实我都比较喜欢NE，因倍的两个结局都觉得还不错。

鲁邦线，毕竟是真·男主，什么叫有恃无恐啊！这条线内容最多，也最仔细，体验是最好的。要打完其他四条线才能解锁鲁邦线有点烦，好在可以快进，每天背单词的时候就打开游戏快进。鲁邦在其他线的表现像是：反正你们都是小老婆，我才是正宫，你们怎么闹我都不怕，随便你们，不仅如此还会帮忙想办法帮帮小老婆们……（不过唯独比较忌惮圣姨太）

总之一旦接受了这个设定（again*3）就能玩下去，五个男主都没有特别对我胃口的，鲁邦线虽然不错，不过考虑到男主原型就是鲁邦三世，还是更喜欢峰不二子那样的女性与他分庭抗礼。游戏整体也是设定更多吧，看着挺厉害的，又是蒸汽伦敦，又是夏尔摩斯。另外，小德这个人物好鸡肋，是为了范这条线而存在的，但因为共通线就出现了所以走其他线时不得不交代他的结局，导致他在其他线里基本就是出现一下一句话交代他要回老家发展了……而我还蛮喜欢他的，他没怎么描写真的很可惜。

总体而言：鲁邦>因倍>圣=弗兰>范，加上感情线的话因倍和圣之间加个壁。
